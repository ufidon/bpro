## Prerequisite
In this course, you may install the programming environment on your personal computer or  a virtual machines with  Ubuntu Mate LTS. To be ready for this course, you need to have
* _Hardware_
  * A laptop or desktop computer with >= 8GB ram, one free USB 3.0+ port
  * A USB 3.0+ Flash drive or external SSD with size >= 256GB if you choose VM, for example:
    * _USB 3.0+ Flash drive_
